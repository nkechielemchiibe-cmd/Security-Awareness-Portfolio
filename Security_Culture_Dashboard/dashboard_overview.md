 Security Culture Power BI Dashboard Overview

This dashboard provides leadership with a real‑time view of human‑factor security performance across the healthcare organization. It consolidates awareness metrics, phishing results, training completion, and incident trends into a single, actionable view.

 Dashboard Purpose
To help executives, managers, and security teams:
- Track improvements in security culture
- Identify high‑risk departments
- Measure the impact of awareness campaigns
- Support data‑driven decision making
- Reduce human‑factor security incidents

 Key Metrics Included
- Phishing simulation failure rate
- Reporting rate (how many staff report phishing)
- Training completion percentages
- Password hygiene compliance
- MFA adoption rate
- Incident response times
- Repeat offenders (de‑identified)
- Department‑level risk scores

 Core Visuals
- Trend lines for phishing failures over 12 months
- Heatmap of high‑risk departments
- Bar chart of training completion by role
- KPI cards for MFA, password compliance, and reporting rate
- Incident timeline showing spikes and patterns
- Awareness campaign impact comparison

 Data Sources
- LMS training data
- Phishing simulation platform
- Identity and access logs
- Incident response tickets
- Monthly awareness engagement metrics

 Intended Audience
- CIO  
- CISO  
- Privacy & Compliance  
- Clinical leadership  
- Security operations teams  

 Value to the Organization
This dashboard helps leadership:
- Understand human‑factor risk at a glance
- Prioritize training and resources
- Demonstrate compliance readiness
- Strengthen overall security culture
